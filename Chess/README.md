Jednostavan ChessGame projekt u Unity game engineu. Implementirano je pomicanje figura, pravilan redoslijed igrača, jedenje figura, davanje šaha i matiranje.
Pijuni se mogu pomicati samo jedno polje unaprijed na prvom potezu i ne mogu promovirati u druge figure na zadnjem polju. Partija se igra lokalno na jednom računalu. Projekt nije u potpunosti dovršen i njegova jedina namjera je bila upoznavanje s Unity enginom.
Kako bi se igra pokrenula, potrebno je otići u Releases i preuzetim .zip datoteku te ju raspakirati. Nakon toga pokrene se .exe file i igra se može igrati.
