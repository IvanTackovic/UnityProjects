Razvijena je jednostavna Catan igra, slična svojoj fizičkoj inačici. Razvijena je u Unity game engineu, koristeći Unity NGO (Netcode for GameObjects), Unity Relay i
blender. Igrica je online multyplayer te se korisnici mogu povezati pomoću koda sa svojih računala na server kojeg host stvori. Implementirano je bacanje igraćih 
kocika, kupnja materijala i struktura, in-game chat, trading system i shop. Projekt je u trajanju i nije još dovršen. Ideja nije bila stvoriti u potpunosti funkcionalnu
igricu, već se upoznati s konceptima online game developmenta. Korišten je i alat za 3d modeliranje bledner u kojem su oblikovane igraće kocke.
Kako bi se igrica pokrenula, potrebno je otići u Releases, preuzeti .zip datoteku i pokrenuti .exe fajl.
